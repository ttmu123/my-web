# 登录页面

一个美观的登录页面，包含手机号验证、密码输入和滑动验证功能。

## 功能特性

- 📱 手机号格式验证
- 🔐 密码输入和验证
- 🎯 滑动验证码
- 👤 用户注册功能
- 🎨 现代化UI设计
- 💬 半透明提示弹窗

## 部署状态

[![Vercel部署状态](https://vercel.com/button)](https://vercel.com/new)

## 本地运行

```bash
# 使用Python启动本地服务器
python -m http.server 3000

# 访问 http://localhost:3000
```

## 技术栈

- HTML5
- CSS3 (渐变、动画、毛玻璃效果)
- JavaScript (ES6+)
- LocalStorage (用户数据存储)

## 部署

本项目已配置为静态网站，可以直接部署到Vercel。
